# SecureCorp DevSecOps Project

SecureCorp is an enterprise-grade backend application designed to demonstrate practical skills in:

- Node.js
- Express.js
- PostgreSQL
- REST APIs
- JWT Authentication
- Role Based Access Control (RBAC)
- Docker
- Terraform
- AWS
- DevSecOps
- OWASP Security Testing

## Project Goals

- Build a production-style backend application
- Learn secure coding practices
- Perform security assessments and pentesting
- Deploy infrastructure using Terraform
- Automate deployments using CI/CD

## Planned Features

### Authentication
- User Registration
- User Login
- Password Hashing
- JWT Authentication

### User Management
- Admin Management
- Employee Management
- Role Based Access Control

### Security
- Audit Logs
- Security Monitoring
- OWASP Top 10 Testing
- Vulnerability Remediation

### Infrastructure
- Docker Containers
- PostgreSQL Database
- AWS Deployment
- Terraform Infrastructure

## Learning Objectives

This project is built as a complete hands-on learning platform for:

- Backend Development
- Cloud Computing
- DevOps
- DevSecOps
- Application Security
- Pentesting
